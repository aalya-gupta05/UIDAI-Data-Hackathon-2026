# UIDAI Data Hackathon 2026 📊

> **Data-Driven Innovation on Aadhaar** — State-wise analysis of Aadhaar biometric, enrolment and demographic datasets across India.

---

## 📋 Project Overview

This project was submitted as part of the **UIDAI Data Hackathon 2026**, focused on deriving meaningful insights from large-scale Aadhaar datasets. The analysis covers state-wise and age-wise usage patterns of Aadhaar services across India.

**Project Title:** State-wise Analysis of Aadhaar Biometric Usage, Enrolment Patterns and Demographic Updates

## 🎯 Objectives

- Clean and merge large-scale Aadhaar datasets
- Perform state-wise and age-wise analysis of enrolment, biometric, and demographic data
- Identify key trends and anomalies across regions
- Visualize insights using graphs for easier interpretation

---

## 🛠️ Tech Stack

- **Language:** Python
- **Libraries:** Pandas, Matplotlib
- **Analysis:** Data cleaning, feature engineering, state-wise aggregation
- **Visualization:** Bar charts, pie charts, geospatial maps

---

## 📊 Datasets Used

- **Biometric Dataset** — Aadhaar biometric activity by state, district, age group
- **Enrolment Dataset** — Aadhaar enrolment counts by state, district, age group
- **Demographic Dataset** — Demographic update records by state and age group

---

## 🔍 Key Findings

- Uttar Pradesh records the highest Aadhaar enrolments, followed by Bihar and Madhya Pradesh
- 96.9% of new enrolments belong to the 0–17 age group, indicating adult Aadhaar saturation
- Demographic updates are significantly higher in the 17+ age group
- Biometric activity shows strong state-wise variation with Tamil Nadu and Andhra Pradesh leading
- Northern and Central India show higher enrolment activity compared to other regions
